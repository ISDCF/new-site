##### ISDCF has worked with Eikon and others* to create open framing charts for the digital cinema industry.

**There are 6 versions of the charts: (selecting any of them will connect to a zip file of the DCP)**

- [2K scope – 2D](https://assets.isdcf.com/ISDCF-Framing_TST-2D_S_XX-XX_US_MOS_2K_ISDCF_20180906_EKN_SMPTE_OV.zip)
- [2K flat – 2D](https://assets.isdcf.com/ISDCF-Framing_TST-2D_F_XX-XX_US_MOS_2K_ISDCF_20180906_EKN_SMPTE_OV.zip)
- [[2K flat/left eye, 2K flat/right eye] – 3D](https://assets.isdcf.com/ISDCF-Framing_TST-3D_F_XX-XX_US_MOS_2K_ISDCF_20180906_EKN_SMPTE_OV.zip)
- [[2K scope/left eye, 2K scope/right eye] – 3D](https://assets.isdcf.com/ISDCF-Framing_TST-3D_S_XX-XX_US_MOS_2K_ISDCF_20180906_EKN_SMPTE_OV.zip)
- [4K scope – 2D](https://assets.isdcf.com/ISDCF-Framing_TST-2D_S_XX-XX_US_MOS_4K_ISDCF_20180906_EKN_SMPTE_OV.zip)
- [4K flat – 2D](https://assets.isdcf.com/ISDCF-Framing_TST-2D_F_XX-XX_US_MOS_4K_ISDCF_20180906_EKN_SMPTE_OV.zip)

All charts have similar characteristics. The chart below is for 2D scope 4K. When the projector is in the proper mode NONE of the red border will be visible. The ideal frame would show all the outside green line on all sides. AT A MINIMUM the yellow line MUST be visible on the screen (if not, you run the risk of on-screen subtitles would not show up). The center white circle can be used to make a brightness reading. The edge targets can be used for focus. The models' color should look life-like. Colorimetry can be measured in the color squares with values shown above the squares.

Downloadable versions of the still frames are located at [https://files.isdcf.com/FramingCharts/](https://files.isdcf.com/FramingCharts/) These are .jpg files and not the highest resolution.
